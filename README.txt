1. Create a new project in your IDE

2. Add all the files exept README.txt into the project

3. Go to your command prompt or terminal and type in pip install pyautogui and press enter/return

4. Also type in pip install opencv-python and press enter/return

5. and type in pip install pillow and press enter/return
